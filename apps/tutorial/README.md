# MiniMax Tutorial (`apps/tutorial`)

Learn MiniMax (mini.nvim) hands-on, one feature per chapter: explain → practice in a
simulated nvim (xterm.js) → quiz. Chapter 1: mini.jump2d.

Part of this repo's *MiniMax-learning* track (vs. its *skill / CLAUDE.md-dev* track).
Stack: pnpm + Vite + React + TS + Tailwind v4 + xterm.js.

## Run
`pnpm install && pnpm dev`
