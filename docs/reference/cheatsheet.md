# MiniMax cheat sheet

## Clipboard (system)

| Key  | Action                          | Source       |
| ---- | ------------------------------- | ------------ |
| `gy` | copy to system clipboard        | mini.basics  |
| `gp` | paste from system clipboard     | mini.basics  |

- Defined by the `mini.basics` preset, not an explicit keymap — see `:h MiniBasics.config.mappings`.
- Config: `~/AppData/Local/nvim-minimax/plugin/30_mini.lua`.
